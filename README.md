# ShopSphere Financial Platform

> Portfólio prático de Engenharia da Qualidade simulando o ciclo completo de qualidade de software em uma plataforma financeira.

---

## Sobre o projeto

A ShopSphere Financial Platform é um **estudo de caso** criado para demonstrar, de forma prática, a atuação de um QA Engineer ao longo de todo o ciclo de qualidade de software.

O projeto simula uma plataforma financeira moderna com funcionalidades de marketplace, pagamentos, carteira digital, crédito pessoal e empréstimos, servindo como base para documentação, testes manuais, automação, métricas e governança de qualidade.

> Este não é um produto comercial, mas um ambiente de aprendizado e demonstração técnica inspirado em cenários reais.

---

## Objetivos

Este projeto demonstra conhecimentos em:

- Cenários BDD
- Testes Manuais

- Testes Mobile
- Testes de APIs

- Automação de Testes

- Governança de Qualidade
- IA aplicada à Engenharia da Qualidade
